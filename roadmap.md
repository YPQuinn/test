# Roadmap

## Final Goal
Build a pluggable, orchestratable, and hierarchical agent workflow using LangGraph, with a web-based UI for management and real-time visualization.

## Milestones

### M1: Project Setup and Basic LangGraph Architecture
*   **Description:** Initialize the Git repository, setup a Python backend (FastAPI + LangGraph) and a modern JS frontend (React/Next.js or Vue). Implement a simple dummy LangGraph workflow.
*   **Status:** Not Started

### M2: Pluggable Agent API
*   **Description:** Develop backend API endpoints to register, unregister, list, and configure modular agents. Establish the hierarchy and orchestration mechanisms via LangGraph state definitions.
*   **Status:** Not Started

### M3: Orchestration Web Interface
*   **Description:** Build the frontend UI allowing users to visually or logically plug in agents, set up their hierarchy (e.g., manager, workers), and start/stop the workflow.
*   **Status:** Not Started

### M4: Real-time Workflow Visualization
*   **Description:** Stream LangGraph execution logs/events to the frontend via WebSockets or SSE so users can monitor the agent workflow in real-time.
*   **Status:** Not Started
